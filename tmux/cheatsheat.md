#### commands
* create new session : `tmux new -s <session name>`
* attach to session by name: `tmux attach -t <session name>`
* enter copy mode (use vim motions): `<tmux leader>[`
* maximize a pane: `<leader>m`
* undo zoom: `<leader>z`
* equalize panes: `<leader>space`
* new tab: `<leader>c`
* switch tab: `<leader><number>`
* rename tab: `<leader>,`
